# Python Socket 编程示例

使用 Python 进行 socket 编程的示例。只讨论 STREAME（比如 TCP） INET（比如 IPv4） socket。

使用了线程处理和异步处理 socket。

## 开始

### 依赖

* Python3

## 运行

在项目目录下，运行 `python server_socket.py` 以启动服务器。

然后在另一个终端的项目目录下，运行 `python client_socket.py` 以启动客户端，然后输入想要传递的信息，
如果一切正常，那么你会看到服务器返回相同的信息。
